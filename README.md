# Project

Full-stack application with React (frontend) + Node.js/Express (backend) + MongoDB Atlas.

## Structure
```
project/
├── frontend/   # React + Redux + Vite
└── backend/    # Node.js + Express + MongoDB
```

## Setup

### Backend
```bash
cd backend
cp .env.example .env   # fill in your values
npm install
npm run dev
```

### Frontend
```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

## Environment Variables
See `.env.example` in each folder.
